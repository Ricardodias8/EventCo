# 01 — Governança EventCo

Esta seção documenta a camada de Governança da plataforma EventCo.

A EventCo é uma plataforma privada que presta serviços tecnológicos para o setor de eventos,
incluindo a operacionalização de regras, fluxos, certificações e compliance definidos por entidades institucionais,
como a ABRAFESTA, respeitando sua autonomia jurídica e institucional.

## Escopo
- Dashboard de Governança EventCo (camada técnica)
- Central de Regras (implementação sistêmica)
- Segmentos e CNAEs (estrutura operacional)
- Normativas (Federal, Estadual, Municipal, ABNT)
- Integração institucional EventCo ↔ ABRAFESTA

## Princípios
- A EventCo não substitui entidades institucionais
- Regras institucionais são definidas por seus respectivos órgãos
- A EventCo implementa, executa e audita regras no ambiente tecnológico
- Nenhuma automação ignora diretrizes institucionais aplicáveis
