# 01 — Governança EventCo

Esta seção documenta a camada de Governança da plataforma EventCo.

## Escopo
- Dashboard de Governança
- Central de Regras
- Segmentos e CNAEs
- Normativas (Federal, Estadual, Municipal, ABNT)
- Integração institucional com a ABRAFESTA

## Princípios
- Nenhuma regra nasce fora do Dashboard de Governança
- Nenhuma operação ignora compliance
- Toda automação possui base normativa
